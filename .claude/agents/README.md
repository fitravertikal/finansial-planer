# Subagents

Curated subagents for building this financial planner. Available in every Claude
Code surface (including web/phone) once merged to the default branch. Invoke by
describing the task/role.

## Selection (13, curated for this project)

**Finance domain**
- `finance-financial-analyst` — modeling, forecasting, scenario analysis
- `finance-fpa-analyst` — budgeting, variance (budget vs actual), rolling forecasts
- `finance-bookkeeper-controller` — categorization, reconciliation, record accuracy

**Engineering**
- `engineering-frontend-developer`, `engineering-backend-architect`,
  `engineering-database-optimizer`, `engineering-rapid-prototyper`,
  `engineering-minimal-change-engineer`, `engineering-code-reviewer`

**Design / Product / Testing**
- `design-ui-designer`, `design-ux-researcher`, `product-manager`,
  `testing-test-automation-engineer`
